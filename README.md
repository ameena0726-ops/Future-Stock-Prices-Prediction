# Future-Stock-Prices-Prediction
This project predicts the next day's stock closing price using historical stock data from Yahoo Finance. It uses features like Open, High, Low, and Volume to train a Linear Regression model and visualizes actual vs predicted prices using Python libraries such as pandas, scikit-learn, and matplotlib.

##Task Objective:
Predict the next day’s stock closing price using historical stock data.

##Dataset:
Source: Yahoo Finance via yfinance Python library
Features: Open, High, Low, Volume
Target: Close price
Example Stocks: Apple (AAPL), Tesla (TSLA)

##Techniques Applied:
Data Handling: pandas
Modeling: Linear Regression 
Visualization: matplotlib (Actual vs Predicted closing prices)

##Key Findings:
The model can roughly predict short-term stock prices.
Price patterns follow trends in Open/High/Low/Volume.
Graphs show predicted prices are close to actual prices but not perfect.
